# Macelectricos

Aplicación móvil desarrollada con FlutterFlow y Firebase para la gestión y registro de infraestructura eléctrica.

## Funcionalidades

- Registro e inicio de sesión
- Registro de postes eléctricos
- Registro de transformadores
- Geolocalización GPS
- Historial de actividades
- Almacenamiento en Firebase

## Tecnologías

- FlutterFlow
- Firebase Authentication
- Cloud Firestore
- GPS
